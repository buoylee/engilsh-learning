

**主从复合句**

# 1. 定语从句

当`定语很简单`时,可以放在`名词前边`, 否则放`后边`.

 从句左边是个名词,  一般就是定语从句.

## 关系代词,副词

- who, whom, which, whose(连词在从句中做定语时), that
- when, where, why, = in/on which, in/at which, for which
-  用于引导从句, 连接主句
- that 省略用法看下方 that 解释.
- `关系代词`充当从句中的句子成分.

## that

- 只能用在`限定性定语从句`中.
- 在从句中做`宾语`时,**that, who, 介 + which的which, (where(place, somewhere, anywhere, everywhere, nowhere少数几个词后时可以省略), when(用于day, year, time等少数几个词后时可以省略), why(通常可换成that或for which))可省略(限定性从句)**.

### 只能用 that

- `先行词`是`不定代词`(some, no, every...)
- `先行词`被`序数词`修饰
- 被'最高级',`the very`,`the only`修饰
- `先行词`前有`who`, `which`等疑问代词
- `先行词`为`人和物`时.
- 当`先行词`做主句的`表语`或`关系代词`做`从句`中的`表语`时

### 不能用that

- 先行词后有`介词`
- `非限定定语从句`
- 从句为`介宾结构`时,  当介词提前到连词前, 不能用that.
- ~~先行词为 one, anyone, those, 人称代词, 不能用. 和that第一条重复~~



### 连词做从句中定语

用 those



### 综合

- 介词后不能加 `who`, `that`. 可以用whom, which.
- of which, 在...中. (例: There are altogether eleven books on the shelf, of which five are mine.)



## 限定/非限定定语从句

- `which`都可以使用
- `that`只能引导限定定语从句.



## 选连词公式

**同名词性从句, 直到从句成分完整.**

|      | 人        | 物                                                           |
| ---- | --------- | ------------------------------------------------------------ |
| 主   | who, that | which, that                                                  |
| 宾   | who(m)    | which, that                                                  |
| 定   | whose     | whose                                                        |
| 状   |           | 除了有3情况, 地点+where, 时间+when, reason+why, 看情况使用`介词+which`(用which试). |



## 其他

1. as

当先行词被`the same, as, such, so`修饰, 连词在从句中做`主, 宾`时, 可以用`as(这里相当于which)`定语引导从句.

```
This is the same pen as I bought yesterday.
Do you have such book as we like?
This is so warm a house as we want to live.
She will marry as healthy a man as she can find.
```

# 3. 状语从句

## 3.1 状语位置

- 状语在句首, 需要用`,`隔开.
- 在`主语`, `谓语`建插入, `插入性状语`, 用`,`隔开("主, 状语, 谓..."). 

## 3.2 地点状语从句

```
I will go where you go (O)
I will go to where you go (X) 主句加了to, 造成to后边需要跟名词, 导致主句不完整.
```

## 3.3 原因状语从句

### 3.3.1. because, since, as, for, now that, in that

For 不能放`句首`表`因为`. now that 只能`XX现在时`. In that 表原因在于, 较书面.

### 3.3.2. because, because of

~ + 句子, ~ of + n./doing

~ of == as a result of == owing to == due to == thanks to 

## 3.4. 让步状语从句

### 3.4.1. although/though/even if(though)/while 尽管

while 1. 当... 2. though 3. 对比(而) 我喜欢...而他喜欢...

in spite of/ despite + n.

## 3.5. 目的状语从句

so that == in order that

so as to do(不能放句首)  ~= in order to do

如果想要 so that 转 so as to do, 需要主从主语一致, 因为 so as to do 没法加主语.
