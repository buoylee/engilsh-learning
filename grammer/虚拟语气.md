

**表`不可能发生`或`与事实相反`的假设句.**

# if从句的虚拟语气

| when | structure                                                |
| ---- | -------------------------------------------------------- |
| 现在 | if did/were, might/could/would/should do                 |
| 过去 | if had done, might/could/would/should have done          |
| 将来 | if did/should do/were to do, might/could/would/should do |
